## Social preview image

The same image is **embedded in the root [README.md](../README.md)** so everyone browsing the repo sees it on the project home page.

For link previews when the repository URL is shared on social platforms, also upload `assets/social-preview.png` manually:

**GitHub:** Settings → General → Social preview → Edit → Upload an image

GitHub does not read social preview from the repository automatically; the README image and this upload are complementary.
