# Mirai Tools

This directory is generated by `mirai --edge`.

Each language gets:

- a small setup file that you edit
- a local SDK copy that you usually do not edit
- a language-specific `README.md`

## Directory Layout

```text
mirai_tools/
├── .env
├── python/
├── swift/
├── typescript/
├── cpp/
├── ue5/
├── go/
├── java/
├── csharp/
├── rust/
├── kotlin/
├── dart/
└── README.md
```

## What To Do Next

1. Open the folder for the language you actually use
2. Read that folder's `README.md`
3. Edit the generated setup file
4. Call the generated init function from your app entry point

## Generate Again

```bash
mirai --edge
```

Generate only some languages (repeat `--lang` or use commas):

```bash
mirai --edge --lang python
mirai --edge --lang rust --lang python
mirai --edge --lang rust,python
mirai --edge --lang swift
mirai --edge --lang typescript
mirai --edge --lang cpp
mirai --edge --lang ue5
mirai --edge --lang go
mirai --edge --lang java
mirai --edge --lang csharp
mirai --edge --lang rust
mirai --edge --lang kotlin
mirai --edge --lang dart
```
